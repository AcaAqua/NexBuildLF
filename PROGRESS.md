# NexBuildLF Project Progress

## 実行フェーズ: UI改善 (2026-04-28)

### 実施内容
1.  **ガントチャートの修正**
    - 工程名列を左側に固定し、タイムラインのみが横スクロールするようにレイアウトを最適化。
    - `z-index` と `position` の整理により、バーが工程名の下に正しく隠れる挙動を実現。
    - 表示サイズ設定（小・中・大）に応じたセル幅・行高さの動的な切り替えに対応。
2.  **表示サイズ設定 (UI Scaling)**
    - `globals.css` にサイズ別のCSS変数を定義。
    - 設定画面から文字サイズや要素の密度を一括変更可能に。
3.  **打ち合わせモードの追加**
    - ナビゲーションを完全に隠した全画面モード `/meeting` を新規作成。
    - 画面上部にプロジェクト情報と「戻る」ボタンを配置。
    - ダッシュボードから1クリックで起動可能な導線を追加。
4.  **Aboutページの追加**
    - アプリのコンセプト、バージョン情報、テスト版であることの免責事項を記載した `/about` ページを作成。
