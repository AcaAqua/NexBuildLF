# User Guide - NexBuildLF

## 1. はじめての方へ
NexBuildLFは、建設現場の工程を直感的に管理するためのアプリです。
ダッシュボードで現在進行中の現場を一目で確認し、ガントチャートで詳細なスケジュールを調整できます。

## 2. 現場の登録
1.  ダッシュボード右上の「＋ 新規現場」ボタンをクリックします。
2.  工事名、種別、住所を入力します。
3.  **メモ欄**には、車両制限や進入経路、近隣住民への配慮事項などを自由に入力してください。

## 3. 工程（タスク）の管理
- **工程の追加**: 現場詳細ページの「＋ 工程追加」ボタンから登録します。
- **多期工事**: 一つの工程に対して複数の期間を設定できます（例：基礎工事の配筋と打設を別日程で登録）。
- **ステータス更新**: 「未着手」「進行中」「完了」の3段階で管理できます。

## 4. データの活用 (CSV連携)
### エクスポート
- ダッシュボードやアーカイブページの「ダウンロード」ボタンから、現在の現場リストをCSV形式で保存できます。
- 日報の作成や、Excelでの集計に活用してください。

### インポート (開発中)
- 外部からCSVを読み込む際は、指定のヘッダー形式に合わせてデータを作成してください。
- 既存のExcel管理表からの移行がスムーズに行えます。

## 5. モバイルでの利用
- スマートフォンでアクセスすると、自動的にモバイル最適化モードに切り替わります。
- 現場での写真撮影や進捗確認に最適です。
