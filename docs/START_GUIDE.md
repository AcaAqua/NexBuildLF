# NexBuildLF スタートガイド 🚀

このガイドでは、NexBuildLF を使い始めるための手順を分かりやすく説明します。

---

## 1. 開発者・管理者向け：ダウンロードと起動方法

パソコンでシステムを管理・改造したり、ローカルで動かしたりする場合の手順です。

### 📥 ダウンロード
GitHub のリポジトリ画面にある **[Code]** ボタンから **[Download ZIP]** を選択してください。
※ `web` フォルダや `sdk` フォルダを個別に落とす必要はありません。**リポジトリ全体**をダウンロードしてください。

### ⚙️ 導入（初回のみ）
1.  ダウンロードしたフォルダを解凍し、ターミナル（コマンドプロンプト）でそのフォルダを開きます。
2.  以下のコマンドを入力して、必要な部品をインストールします。
    ```bash
    npm install
    ```

### ▶️ 起動
以下のコマンドでアプリが立ち上がります。
```bash
npm run dev
```
起動後、ブラウザで `http://localhost:3000` を開くと管理画面が表示されます。

## 2. ユーザー向け：スマホへのインストール方法（PWA）

NexBuildLF をスマホで「本物のアプリ」として使いたい場合の手順です。**特別なダウンロードは不要です。**

### 📱 iPhone (Safari) の場合
1.  Safari で NexBuildLF の URL を開きます。
2.  画面下部の **[共有ボタン]**（上向き矢印のアイコン）をタップします。
3.  メニューをスクロールし、**[ホーム画面に追加]** をタップします。
4.  右上の **[追加]** を押すと、ホーム画面に NexBuildLF のアイコンが現れます。

### 🤖 Android (Chrome) の場合
1.  Chrome で NexBuildLF の URL を開きます。
2.  画面右上の **[メニューボタン]**（点3つのアイコン）をタップします。
3.  **[アプリをインストール]** または **[ホーム画面に追加]** をタップします。
4.  確認画面で **[インストール]** を押すと、アプリ一覧に NexBuildLF が追加されます。

---

## 3. オフラインでの利用について
一度スマホのホーム画面に追加すれば、電波のない現場でもアイコンをタップするだけでアプリが起動します。
現場で入力したデータはスマホ内に保存され、次にネットに繋がったときやブラウザを開いた際にも維持されます。
