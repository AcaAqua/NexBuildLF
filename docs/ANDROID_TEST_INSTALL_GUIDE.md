# NexBuildLF Androidテスト版 インストール手順

## 注意

このアプリは社内・知人向けのテスト版です。
Google Playストアでの正式公開前のため、Android端末でインストールする際に「提供元不明のアプリ」という警告が表示される場合があります。

## インストール手順

1. 提供されたリンクから **APKファイル (`app-debug.apk`)** をダウンロードします。
2. Android端末の「ダウンロード」またはファイル管理アプリで、ダウンロードしたAPKファイルを開きます。
3. セキュリティ警告（「提供元不明のアプリをインストールできません」等）が出た場合は、「設定」をタップし、「この提供元のアプリを許可」をオンにします。
4. 「インストール」をタップして完了を待ちます。
5. ホーム画面またはアプリ一覧から「NexBuildLF」を起動します。

## テストしてほしい項目

以下の操作が一通り問題なく行えるかをご確認ください。

- [ ] アプリが正常に起動し、画面が表示されるか
- [ ] ダッシュボードから「新規現場」を登録できるか
- [ ] 登録した現場を開き、工程（タスク）を追加・編集できるか
- [ ] インターネットに繋がっていない状態（オフライン・機内モード）でもアプリが動作し、データが保存されるか
- [ ] アプリを一度終了（タスクキル）し、再起動してもデータが消えずに残っているか
- [ ] 設定画面から「バックアップを作成」し、スマホ内にファイルとして保存できるか
- [ ] 設定画面から「バックアップを復元」し、データが元通りに戻るか
