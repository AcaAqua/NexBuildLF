## 注意

このアプリは社内・知人向けのテスト版です。
Google Playストアでの正式公開前のため、Android端末でインストールする際に「提供元不明のアプリ」という警告が表示される場合があります。

## インストール手順

1. 提供されたリンクまたはファイル共有から **APKファイル (`NexBuildLF-debug-20260607.apk`)** をダウンロードします。
2. Android端末の「ダウンロード」またはファイル管理アプリで、ダウンロードしたAPKファイルを開きます。
3. セキュリティ警告（「提供元不明のアプリをインストールできません」等）が出た場合は、「設定」をタップし、「この提供元のアプリを許可」をオンにします。
4. 「インストール」をタップして完了を待ちます。
5. ホーム画面またはアプリ一覧から「NexBuildLF」を起動します。
